## Input / Parameter

| Name | Description | Input Type | Default | Options | Required |
| ------ | ------ | ------ | ------ | ------ | ------ |
| value | The value to be checked. | String/Text | - | - | Yes |

## Output

| Description | Output Type |
| ------ | ------ |
| Returns true if value is an email otherwise false. | Boolean |
